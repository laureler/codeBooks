  • {{item}}

    • (item , index) in array

  • {{index}}-{{item}}

    • (val , key) in array

  • {{key}}-{{val}}

    • (val,key,index) in array

  • {{index}}-{{key}}-{{val}}

    • (val , key) in object

  • {{key}}-{{val}}

    • (item , index) in object

  • {{index}}-{{item}}

    • (val,key,index) in object

    index-key-val
  • {{index}}-{{key}}-{{val}}

    • 你可能会注意到，array并没有对应的 value,key,index模式，取而代之的是 value,index。
    vue官网也解释得很清楚，<div v-for="(value, key, index) in object">
    只有object作为数据源的时候，才会有对应的,val,key,index模式
